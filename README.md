## 1. 第一课
> git命令
>> * git branch *查看git分支*
>> * git branch [name] *创建分支*
>> * git branch -a *查看分支(包含远程分支)*
>> * git checkout [name] *切换分支*
>> * git checkout -b [name] *创建并选择分支*
>> * git push *推送,远程无该分支时命令行会出现推送分支的命令*
>> * git status *查看状态*
>> * git add . *暂存所有文件(存储在本地仓库)*
>> * git commit -m '这里是当前提交的描述' *提交*
>> * git push *推送*
>> * git pull *拉取*
